# 资料分类、证据等级与引用边界

## 资料分类

| source_type | 分类 | 主要用途 |
| --- | --- | --- |
| `legal_regulation` | 法律法规与规章 | 合规依据、处置边界、重大隐患人工判定参考 |
| `standard_spec` | 国家标准、行业标准、工程建设规范 | 监测监控、抽采、通风、防突、报警、断电、复电、验收 |
| `policy_notice` | 政策文件与监管通知 | 监管重点、企业自查、系统验收、整改建议 |
| `case_report` | 事故案例库 | 警示教育、相似性参考、风险链路复盘、报告引用 |
| `enterprise_document` | 企业制度与现场资料 | 企业落地闭环、现场审核、制度执行 |
| `equipment_manual` | 传感器与设备资料 | 数据质量解释、测点边界、量程、单位、调校周期 |
| `cnki_literature` | 中文论文资料 | 机理解释、模型解释、工程经验参考 |
| `wos_literature` | 英文论文资料 | 国际研究、Review、工程应用、模型方法 |
| `data_model_evidence` | 数据样本与模型评估资料 | 数据模型模块、质量报告、63 通道来源、验证报告 |

## 证据等级

| 等级 | 定义 | 合规依据 | AI 引用优先级 |
| --- | --- | --- | --- |
| L1 | 法律、行政法规、部门规章 | 是 | 最高 |
| L2 | 国家标准、行业标准、工程建设标准 | 是 | 高 |
| L3 | 官方事故调查报告、监管通报、警示教育 | 有条件，需要人工审核 | 中 |
| L4 | 企业制度、现场资料、设备说明书 | 有条件，仅限本企业或本场景 | 中 |
| L5 | 学术论文、教材、技术文章 | 否 | 低 |

AI 后续引用必须优先使用 L1/L2，其次 L3/L4，最后 L5。论文只能支撑机理、模型解释、指标候选和工程经验，不能替代法规标准。事故案例只能支撑警示教育、相似性参考和复盘，不能替代当前现场判断。

## 人工审核边界

L1/L2 存在版本、废止、修订或施行日期不确定时必须审核。L3 事故案例用于事故原因、责任认定、整改措施时必须审核。L4 企业内部资料含人员、位置、生产计划、设备编号、图纸或敏感经营信息时必须审核并判断脱敏。L5 论文结论用于解释现场处置、指标阈值或模型有效性时必须审核。

## 禁止误用

- 不得把论文结论写成法规要求。
- 不得把事故案例相似性写成事故必然性。
- 不得用案例替代现场复核、调度研判和人工法规审核。
- 不得把 `physics_constrained` 或生成估计指标写成真实传感器。
- 最新业务通道口径为 63 个通道，数据模型资料必须保留通道来源标记。
